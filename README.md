# Hex UTF-8 Converter

A black and white themed web tool that decodes hex-encoded UTF-8 input into text, with optional JSON formatting and highlighting.

## Run locally

Open `index.html` in a browser.

## Deploy to GitHub Pages

1. Create a new GitHub repository and push this folder.
2. In GitHub, go to Settings -> Pages.
3. Set "Build and deployment" to "Deploy from a branch".
4. Select the `main` branch and `/root` folder.
5. Save. GitHub will provide a public URL.

## Notes

- The input accepts hex with or without spaces or `0x` prefixes.
- Enable Auto-format JSON to pretty-print and highlight valid JSON output.
